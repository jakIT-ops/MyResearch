About

This App is a demo for the Microsoft Computer Vision API.

MS Computer Vision gives access to advanced algorithms for image processing and returning information. In this App you can provide a URL that is analyzed by the algorithm depending on the inputs and user choices.

Functionalities

To start you will have to add your subscription_key and endpoint on these two by creating a MS Computer Vision resource on Azure.

Following are the functionalities provided in the App:

1. Optical Character Recognition (OCR):
